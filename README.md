# Miniaturas

Generador paramétrico de miniaturas SVG para muebles de cocina.

## App básica de captura de medidas

Se añadió una app web sencilla (`index.html`) que pide estos campos:

- Nombre
- Ancho (en mm)
- Alto (en mm)
- Fondo (en mm)

### Cómo usarla

1. Ejecuta un servidor estático en la raíz del proyecto:

```bash
python3 -m http.server 8000
```

2. Abre en el navegador:

```text
http://localhost:8000
```

3. Completa el formulario y pulsa **Guardar datos**.
